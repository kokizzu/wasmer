1 # Wasmer Runtime C API
2 
3 [Wasmer] is a standalone WebAssembly runtime for running WebAssembly [outside of the browser](https://webassembly.org/docs/non-web/), supporting [WASI](https://github.com/WebAssembly/WASI) and [Emscripten](https://emscripten.org/).
4 
5 The Wasmer Runtime C API exposes a C and a C++ API to interact
6 with the Wasmer Runtime, so you can use WebAssembly anywhere.
7 
8 [Wasmer]: https://github.com/wasmerio/wasmer
9 [WebAssembly]: https://webassembly.org/
10 
11 # Usage
12 
13 The shared object files and the headers will
14 be automatically available **inside the Wasmer installed path**.
15 
16 > Please check the following docs to see how to [install Wasmer in your system](https://github.com/wasmerio/wasmer#1-install-wasmer).
17 
18 The C ([`wasmer.h`][wasmer_h]) and C++ ([`wasmer.hh`][wasmer_hh]) header
19 files can be found in the Wasmer `include` directory:
20 
21 ```bash
22 wasmer config --includedir
23 ```
24 
25 The runtime shared libraries (`.so`, `.dylib`, `.dll`) can be found in the Wasmer
26 `lib` directory:
27 
28 ```bash
29 wasmer config --libdir
30 ```
31 
32 > Note: You can also download the libraries or header files directly
33 from [Wasmer release page].
34 
35 The full C API documentation can be found here:
36 https://wasmerio.github.io/wasmer/c-api/
37 
38 Here is a simple example to use the C API:
39 
40 ```c
41 #include <stdio.h>
42 #include "wasmer.h"
43 #include <assert.h>
44 #include <stdint.h>
45 
46 int main()
47 {
48  // Read the Wasm file bytes.
49  FILE *file = fopen("sum.wasm", "r");
50  fseek(file, 0, SEEK_END);
51  long len = ftell(file);
52  uint8_t *bytes = malloc(len);
53  fseek(file, 0, SEEK_SET);
54  fread(bytes, 1, len, file);
55  fclose(file);
56 
57  // Prepare the imports.
58  wasmer_import_t imports[] = {};
59 
60  // Instantiate!
61  wasmer_instance_t *instance = NULL;
62  wasmer_result_t instantiation_result = wasmer_instantiate(&instance, bytes, len, imports, 0);
63 
64  assert(instantiation_result == WASMER_OK);
65 
66  // Let's call a function.
67  // Start by preparing the arguments.
68 
69  // Value of argument #1 is `7i32`.
70  wasmer_value_t argument_one;
71  argument_one.tag = WASM_I32;
72  argument_one.value.I32 = 7;
73 
74  // Value of argument #2 is `8i32`.
75  wasmer_value_t argument_two;
76  argument_two.tag = WASM_I32;
77  argument_two.value.I32 = 8;
78 
79  // Prepare the arguments.
80  wasmer_value_t arguments[] = {argument_one, argument_two};
81 
82  // Prepare the return value.
83  wasmer_value_t result_one;
84  wasmer_value_t results[] = {result_one};
85 
86  // Call the `sum` function with the prepared arguments and the return value.
87  wasmer_result_t call_result = wasmer_instance_call(instance, "sum", arguments, 2, results, 1);
88 
89  // Let's display the result.
90  printf("Call result: %d\n", call_result);
91  printf("Result: %d\n", results[0].value.I32);
92 
93  // `sum(7, 8) == 15`.
94  assert(results[0].value.I32 == 15);
95  assert(call_result == WASMER_OK);
96 
97  wasmer_instance_destroy(instance);
98 
99  return 0;
100 }
101 ```

109 ## pkg-config
110 
111 The Wasmer binary ships with an utility tool that outputs config
112 in the `pkg-config` format.
113 
114 You can use it like:
115 
116 ```bash
117 wasmer config --pkg-config > $PKG_CONFIG_PATH/wasmer.pc
118 ```
